# 📊 Revenue and Team Performance Analysis (SQL Project)

## 🧩 Business Problem
The company wants to analyze:
- Total revenue
- Target vs Achievement %
- Regional performance
- Salesperson performance
- Operational delays impact
- Manager-wise revenue contribution

## 🛠 Tools Used
- PostgreSQL and power bi 
- SQL (JOIN, GROUP BY, Aggregations, KPI Analysis)
-power bi (visuals ,dax , relationship)

## 📌 Key Insights Generated
- Overall achievement %
- Top & Bottom Salespersons
- Highest Revenue Region
- Delayed Orders Analysis
- Manager-Level Revenue
